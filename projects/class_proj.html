RISC-V ALU & CPU

Devleoped a RISC-V ALU and then CPU for 32 bit systems using logisim, a logical circuit simulator. Developed testing scripts to automatically test inputs to the circuit for both the whole system and smaller components. The overall structure of the CPU is as follows. An instruction is fetched from memory, decoded, executed and results are loaded into memory or written back to registers as needed. The following instructions are supported: ADD, SUB, AND, SLT, SLL, SRA, ADDI, ANDI, LW, LB, SW, SB, LUI, BEQ.

Cache Simulator

Created a C program which simulates the behavior of a cache under different protocols (VI and MSI). The simulator will be configurable so that it can model the behavior of a variety of cache configurations such as different block sizes and cache associativity. The simulation also records different statistics such as traffic in and out of the cache and miss/hit rates for write-through and write-back caches. The simulator reads a memory trace mimicking the loads/stores of a program.

RML Interpreter

Devleoped an interepreter for an OCaml like language called RML (Robot Meta-Language).

Froggit

Devleoped a clone of the game frogger in python. The project involved loading the levels, implementing game logic and state, and lastly adding in basic annimation and sound. The graphical objects (different tiles), backgrounds, fonts, and animation frames were all given to students for this assignment. To play the game, you need to have the Cornell Extensions Python package installed for the correct package configurations. This can be found on the CS1110 Fall 2020 webpage.
