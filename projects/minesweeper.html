Goal

We aim to make a minesweeper game in OCaml that is playable in the terminal. The game can be installed off of our github.

Process

We implement the game game logic in a purely fuctional manner and implement unit and integration tests for each component of the game to ensure proper functionality.

Highlights

The game is developed in a purely functional manner, meaning that no mutable states are used and each function only depends on its input.
